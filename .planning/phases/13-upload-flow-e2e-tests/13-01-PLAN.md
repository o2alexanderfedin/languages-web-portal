Methods:
- `goto(params?: string)` — Navigate to `/demo` with optional query params (e.g., `tool=java-verification`)

- `waitForUploadSuccess(timeout?: number)` — Wait for `uploadSuccess` to be visible

- `getSuccessText()` — Return text content of `uploadSuccess`
- `getErrorText()` — Return text content of `uploadError`

- `clickUploadAnother()` — Click the Upload Another button
- `clickTryAgain()` — Click the Try Again button
- `dragAndDropFile(filePath: string)` — Use Playwright `page.dispatchEvent` on the upload zone to simulate drag-and-drop. Use `DataTransfer` approach: read file, create drag events (dragenter, dragover, drop) on the upload zone element. Implementation:
  ```typescript
  async dragAndDropFile(filePath: string) {
    const buffer = fs.readFileSync(filePath);
    const dataTransfer = await this.page.evaluateHandle(
      (data) => {
        const dt = new DataTransfer();
        const file = new File([new Uint8Array(data)], 'sample.zip', { type: 'application/zip' });
        dt.items.add(file);
        return dt;
      },
      [...buffer]
    );
    await this.uploadZone.dispatchEvent('drop', { dataTransfer });
  }
  ```
